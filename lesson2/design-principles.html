White Space

Lyra

Lyra.com

Lyra's website is one of the best which illustrate utilisation of white space and clean design principle. Both padding and margins are used to create a design with not only visual appeal but one whivh is responsive. In the example given from the screenshot above we see adequate padding in the entire content of the page as well as the "explore options button" and the menu section. Elements are adequately spaced using margins

Alignment

Squid Ink

squid-ink.com

Squid Ink website illustrates alignment design principles. It is one of the examples of left-alignmet which is common in languages which read left to right. It also utilises the common F-design. This design gives a sense of balance which confers it visual appeal

Repetition

Chartbeat

chartbeat.com

Chartbeat exhibits repetition of content and elements on which shows not only cohesion but also serves as a marketing strategy for the business Charbeat repeats links and images which show the business' products. The structure, layout, schemes, colors, styles, and fonts are repeated throughout the site. This enhances the user experience.
